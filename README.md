# ToDo App V6

Features:
- Add Task
- Edit Task
- Delete Task
- Mark as Completed
- Local Storage
- Search Tasks
- Filter Tasks
- Dark Mode
- Progress Bar
- Task Counter
- Due Date
- Priority Levels
- Categories
- Sort by Priority
- Sort by Due Date


## An advanced ToDo application built with HTML, CSS and JavaScript. It supports task management with priorities, categories, due dates, search, sorting, dark mode and local storage.
